== Description ==

Add a button that allows visitors to scroll smoothly to the top of the page. 
Customizable options via the admin panel (Button Position,size, color, transparency, icon image etc...)

== Installation ==

1. Upload plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to 'Settings'->'Go-To-Top Button' to customize button style and position
